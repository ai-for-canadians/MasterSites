# Content Review Checklist

Complete for each piece of content before publishing.

## Accuracy
- [ ] All claims verified
- [ ] Statistics have sources
- [ ] Sources are credible
- [ ] Information is current

## Evidence
- [ ] Claims have citations
- [ ] Evidence ratings applied
- [ ] Study types identified
- [ ] Limitations noted

## Canadian Context
- [ ] Canadian spelling
- [ ] Canadian resources
- [ ] Provincial variations noted
- [ ] Prices in CAD

## Tone
- [ ] Matches brand voice
- [ ] Not condescending
- [ ] Acknowledges uncertainty
- [ ] Empowering tone

## Technical
- [ ] Grammar correct
- [ ] Links work
- [ ] Images have alt text
- [ ] Headings logical
