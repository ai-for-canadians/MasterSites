# Pre-Build Checklist

Complete before starting any new site build.

## Planning
- [ ] User value clearly articulated
- [ ] Target audience defined
- [ ] Key user journeys mapped
- [ ] Success metrics identified

## Technical
- [ ] Data structure planned
- [ ] Pages listed with purposes
- [ ] Features prioritized (MVP vs future)
- [ ] Dependencies identified

## Content
- [ ] Seed data requirements defined
- [ ] Content sources identified
- [ ] Canadian context needs noted

## Design
- [ ] Design system reviewed
- [ ] Site-specific variations documented
- [ ] Mobile experience planned

## Prompt
- [ ] Follows template structure
- [ ] References design system
- [ ] Includes acceptance criteria
