# Pre-Launch Checklist

Complete before deploying to production.

## Performance
- [ ] Lighthouse Performance > 90
- [ ] Lighthouse Accessibility > 90
- [ ] Lighthouse SEO > 90
- [ ] No console errors
- [ ] Page load < 3 seconds

## SEO
- [ ] All pages have unique titles
- [ ] All pages have meta descriptions
- [ ] Open Graph images set
- [ ] Sitemap generated
- [ ] robots.txt correct
- [ ] Structured data validates

## Security
- [ ] No exposed secrets
- [ ] Environment variables set
- [ ] RLS enabled on tables
- [ ] Auth working (if applicable)

## Content
- [ ] All content reviewed for accuracy
- [ ] Sources verified
- [ ] No placeholder content
- [ ] Disclaimers in place

## Cross-Browser
- [ ] Chrome desktop
- [ ] Safari desktop
- [ ] Firefox desktop
- [ ] Chrome mobile
- [ ] Safari mobile (iOS)

## Final
- [ ] Someone else has reviewed
- [ ] Analytics installed
- [ ] Error tracking active
